
# VA Online Scheduling - Content

## Content templates

### Scheduling pages

Scheduling pages follow this content structure:

* VA.gov header
* VAOS breadcrumb
* Form title
* Page header
* Page description
  * Optional - may be an input label or regular text
* Page content
  * Variable - use language defined in VA.gov design system for standard components
* Progress buttons
* VAOS footer title
* VAOS footer content
* VA.gov footer
