### Log into staging and check out the landing page (requires access to the VA network)
Follow the steps below to find user credentials for logging in:

- Log into the VA network 
- Log into Slack from a browser on the VA virtual desktop and navigate to this ticket so you can copy/paste
- Open a new browser tab and navigate to the Test User Dashboard https://tud.vfs.va.gov/?page=1
- Log in using your GitHub account
- Filter the results to show users with Healthcare and ID Type ID.me.
- Find user whose email contains 377
- Go to https://staging.va.gov
- Clik _Sign In_ in the main nav
- Select the ID.me option
- Enter the test user's email address and password
- When prompted to receive a code by phone:
    -  Make sure _Text me_ is selected
    -  Leave the phone number field as-is
    -  Click the _Continue_ button
-  The screen will refresh, and a six-digit code will be displayed.  Press _Continue_.
-  Once logged in, click the _My HealtheVet_ link at the far-right in the main nav
-  You should see the My HealtheVet on VA.gov landing page load
-  As of April 3, 2023, the links on this page go to tools and content on the "classic" My HealtheVet (MHV) web site or VA.gov
-  As MHV applications and content is integrated into VA.gov, the links on this landing page will be updated to point to the new experience
