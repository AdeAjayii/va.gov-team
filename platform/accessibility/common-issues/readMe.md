# Common Accessibility Issues

- [Headings should be consistently used](https://github.com/department-of-veterans-affairs/va.gov-team/issues/25736)
- [Text styled like headings must be marked up as headings](https://github.com/department-of-veterans-affairs/va.gov-team/issues/18246)

## Focus

* [Focus should not be lost on the page](https://github.com/department-of-veterans-affairs/va.gov-team/issues/23968)
* [Focus outline should be removed from non-linked heading](https://github.com/department-of-veterans-affairs/va.gov-team/blob/master/platform/accessibility/common-issues/focus-outline-should-be-removed-from-non-linked-heading.md)

## Screen Readers

* [Content being added or removed from the page must announce a change to assistive technology using `aria-live`](https://github.com/department-of-veterans-affairs/va.gov-team/issues/19660)
* [Important unfocusable text sandwiched between form fields must be announced to assistive technology using aria techniques](https://github.com/department-of-veterans-affairs/va.gov-team/issues/21478)
* [Visual groupings must be clear to screen reader users using semantic HTML or aria techniques](https://github.com/department-of-veterans-affairs/va.gov-team/issues/21478)
* [Groups of form elements MUST have group labels (e.g. `<fieldset>` and `<legend>`)](https://github.com/department-of-veterans-affairs/va.gov-team/issues/18569)
* [Decorative content must be implemented in a way it can be ignored by assistive technology](https://github.com/department-of-veterans-affairs/va.gov-team/blob/master/platform/accessibility/common-issues/decorative-content.md)
* [Links shouldn't open in a new tab or window unless absolutely required, otherwise notify the user with (open in new tab)](https://github.com/department-of-veterans-affairs/va.gov-team/issues/25112)
* [Page navigation should use semantic landmarks and unique labels](https://github.com/department-of-veterans-affairs/va.gov-team/issues/18398)
* [Purely presentational content should not be announced to screen reader users](https://github.com/department-of-veterans-affairs/va.gov-team/issues/18438)

## Cognition
